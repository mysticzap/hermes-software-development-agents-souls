# 创建 Vim 基本常用命令的 Markdown 文档

## 一、模式切换

### 1. 进入不同模式
- **普通模式**：按 `Esc` 键（默认进入的模式）
- **插入模式**：
  - `i`：在光标前插入
  - `I`：在行首插入
  - `a`：在光标后插入
  - `A`：在行尾插入
  - `o`：在当前行下方新建一行并插入
  - `O`：在当前行上方新建一行并插入
- **可视模式**：
  - `v`：字符可视模式
  - `V`：行可视模式
  - `Ctrl+v`：块可视模式
- **命令模式**：按 `:` 进入

## 二、光标移动

### 1. 基本移动
- `h`：左移
- `j`：下移
- `k`：上移
- `l`：右移
- `w`：移动到下一个单词开头
- `b`：移动到上一个单词开头
- `e`：移动到单词末尾
- `0`：移动到行首
- `$`：移动到行尾
- `gg`：移动到文件开头
- `G`：移动到文件末尾
- `:n`：跳转到第 n 行（如 `:10` 跳转到第 10 行）

### 2. 翻页
- `Ctrl+f`：向下翻一页
- `Ctrl+b`：向上翻一页
- `Ctrl+d`：向下翻半页
- `Ctrl+u`：向上翻半页

## 三、文本编辑

### 1. 删除操作
- `x`：删除光标所在字符
- `dd`：删除当前行
- `dw`：删除一个单词
- `d$` 或 `D`：删除到行尾
- `d0`：删除到行首
- `dG`：删除到文件末尾
- `dgg`：删除到文件开头

### 2. 复制与粘贴
- `yy`：复制当前行
- `yw`：复制一个单词
- `y$`：复制到行尾
- `p`：在光标后粘贴
- `P`：在光标前粘贴

### 3. 撤销与重做
- `u`：撤销上一次操作
- `Ctrl+r`：重做上一次撤销

### 4. 替换操作
- `r`：替换单个字符（按 r 再按新字符）
- `R`：进入替换模式（连续替换）
- `cw`：修改一个单词（删除并进入插入模式）
- `cc`：修改整行
- `c$`：修改到行尾

## 四、查找与替换

### 1. 查找
- `/pattern`：向前查找 pattern
- `?pattern`：向后查找 pattern
- `n`：查找下一个匹配项
- `N`：查找上一个匹配项
- `*`：查找当前光标下的单词

### 2. 替换
- `:s/old/new`：替换当前行第一个 old 为 new
- `:s/old/new/g`：替换当前行所有 old 为 new
- `:%s/old/new/g`：替换全文所有 old 为 new
- `:%s/old/new/gc`：替换全文，每次替换前确认

## 五、保存与退出

### 1. 保存文件
- `:w`：保存文件
- `:w filename`：另存为 filename
- `:wq` 或 `ZZ`：保存并退出
- `:x`：保存并退出（仅当文件有修改时）

### 2. 退出
- `:q`：退出（无修改时）
- `:q!`：强制退出，不保存修改
- `:qa`：退出所有窗口
- `:qa!`：强制退出所有窗口

## 六、窗口与分屏

### 1. 分屏操作
- `:sp` 或 `:split`：水平分屏
- `:vsp` 或 `:vsplit`：垂直分屏
- `Ctrl+w w`：在窗口间切换
- `Ctrl+w h/j/k/l`：向左/下/上/右切换窗口
- `Ctrl+w =`：使所有窗口等宽等高
- `Ctrl+w +`：增加窗口高度
- `Ctrl+w -`：减少窗口高度
- `:q`：关闭当前窗口

### 2. 标签页
- `:tabnew`：新建标签页
- `:tabn` 或 `gt`：下一个标签页
- `:tabp` 或 `gT`：上一个标签页
- `:tabclose`：关闭当前标签页

## 七、其他实用命令

### 1. 缩进
- `>>`：向右缩进当前行
- `<<`：向左缩进当前行
- `=`：自动缩进当前行
- `gg=G`：自动缩进整个文件

### 2. 显示行号
- `:set number` 或 `:set nu`：显示行号
- `:set nonumber` 或 `:set nonu`：隐藏行号
- `:set relativenumber`：显示相对行号

### 3. 语法高亮
- `:syntax on`：开启语法高亮
- `:syntax off`：关闭语法高亮

### 4. 搜索高亮
- `:set hlsearch`：高亮搜索结果
- `:set nohlsearch`：取消高亮
- `:nohlsearch` 或 `:noh`：临时取消高亮

## 八、组合命令示例

### 1. 删除多行
- `5dd`：删除 5 行
- `d5j`：删除当前行及下面 5 行

### 2. 复制多行
- `5yy`：复制 5 行
- `y5j`：复制当前行及下面 5 行

### 3. 移动多行
- `5>>`：向右缩进 5 行
- `5<<`：向左缩进 5 行

## 九、配置文件

### 1. 编辑配置文件
```bash
vim ~/.vimrc
```

### 2. 常用配置示例
```vim
" 显示行号
set number

" 语法高亮
syntax on

" 自动缩进
set autoindent

" Tab 转换为空格
set expandtab
set tabstop=4
set shiftwidth=4

" 搜索时忽略大小写
set ignorecase
set smartcase

" 高亮搜索结果
set hlsearch

" 显示命令
set showcmd

" 显示状态栏
set laststatus=2
```

## 十二、实用技巧

### 1. 快速编辑技巧
- **重复操作**：在普通模式下按 `.` 重复上一次编辑操作
- **宏录制**：
  - `q` + 寄存器名：开始录制宏
  - 执行操作
  - `q`：停止录制
  - `@` + 寄存器名：执行宏
- **标记位置**：
  - `m` + 字母：标记当前位置
  - `'` + 字母：跳转到标记位置

### 2. 批量操作
- **多行注释**：
  1. 按 `Ctrl+v` 进入块可视模式
  2. 选择要注释的行
  3. 按 `I` 进入插入模式
  4. 输入注释符号（如 `#`）
  5. 按 `Esc`，所有选中行都会添加注释
- **多行取消注释**：
  1. 按 `Ctrl+v` 进入块可视模式
  2. 选择注释符号
  3. 按 `d` 删除

### 3. 文件操作
- `:e filename`：打开新文件
- `:bn`：切换到下一个缓冲区
- `:bp`：切换到上一个缓冲区
- `:bd`：关闭当前缓冲区
- `:ls`：列出所有缓冲区
